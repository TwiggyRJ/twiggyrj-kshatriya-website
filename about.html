About Us:
At Kshatriya

Who Are Kshatriya?

Kshatriya is an alias for the creator of this website Aaron Fryer. It all started when Aaron had to create a brand name and a logo for himself or a company for his DES400: Digital Creatvity Brand Exercise Assignment. Kshatriya is a student in his first year of university studying Web Design & Development and creates HTML, CSS, PHP documents to improve his web development skills.

He is currently expanding onto the App development area, first by developing Windows Phone and Windows 8 applications and then moving onto different platforms when he is fully comfortable in C#. His current aims are to re-learn JavaScript and get a better understanding of it an improve is native development skills on the Windows Platform.

Our current aim is to create and provide a social web service that is open to outside use and is extensible.

How it was Created

Kshatriya was created for a brand exercise, though the brand name and logo has been kept since the creation and completion of the brand exercise. The name Kshatriya came from an anime called Mobile Suit Gundam: Unicorn where the logo was based on the mobile suit of the same name. The name was chosen because it was unique and the only other notable reference to it was a film from the 1980's that didn't gain any fame. This enabled me to use the name because the name was still fresh and nothing of too noticable was using it. The logo was based on the mobile suits head and flattened and simplified to express the design style of the creator Aaron Fryer.

Our Design Scheme

The design scheme here at Kshatriya is as follows:

  • The main colour for Kshatriya is green in the hex colour of: #336633
  • Main body text is of the hex colour of: #333 and uses the Open Sans font face with a weight of 300
  • Page headers are of the size of 3.5em and use the main colour of choice which is: #336633
  • Form text fields have a glow of the secondary colour of: #ff99cc

This design scheme is to ensure that this site follows the creators design style.

How We Develop

We use the spiral methodology and develop websites incrementally. This allows us to continuously develop the pages of the website with new requirements. Developing this website has taken many revisions and has been subject to many different design styles, we have now settled on a contemporary flat design style.

Mobile Friendly & Responsive Design

This website uses percentages, media queries and the responsive design grid system to create a website that responds to the screen size so that content is never lost. This enables users from any device to get a quality experience.
